# FX Visualizer - Keyboard Shortcuts Reference

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Arrow Keys` | Move nodes | Move selected nodes in 3D space |
| `Shift + Arrow` | Pan camera | Move camera view |
| `Ctrl + Scroll` | Zoom | Zoom in/out |
| `Mouse Wheel` | Zoom | Alternative zoom control |
| `Right Click + Drag` | Pan | Pan camera view |
| `Middle Click + Drag` | Rotate | Rotate 3D view |
| `F` | Focus | Focus camera on selected node |
| `A` | Fit all | Fit all nodes in view |
| `0` | Reset camera | Return to default camera position |
| `1` | Default view | Apply "Default" camera preset |
| `2` | Top view | Apply "Top" camera preset |
| `3` | Side view | Apply "Side" camera preset |
| `4` | Front view | Apply "Front" camera preset |
| `5` | Layers view | Apply "Layers" camera preset |
| `6-9` | Custom views | Apply custom camera presets (if defined) |

## Contextual Shortcuts

These shortcuts work in specific contexts:

### Inspector Panel
- `Ctrl + E`: Expand all properties
- `Ctrl + Shift + E`: Collapse all properties
- `Ctrl + C`: Copy property value
- `Enter`: Edit property value

### Console Panel
- `Ctrl + L`: Clear console
- `Ctrl + F`: Filter logs
- `Up/Down`: Navigate log history
- `Ctrl + K`: Clear filter

### Timeline Panel
- `Space`: Play/Pause
- `Left/Right`: Navigate frames
- `+/-`: Adjust playback speed

## Tips

1. Hold `Shift` while performing an action to modify its behavior
2. Most shortcuts can be customized in Settings
3. Press `?` anytime to see this shortcuts reference
4. Hover over buttons to see their shortcuts in tooltips

---

**Pro tip**: Learn a few shortcuts at a time. Start with navigation and selection, then add more as you become comfortable.
