# FXD Quantum Visualizer - Keyboard Shortcuts Reference

Quick reference for all keyboard shortcuts in the FXD Quantum Desktop Visualizer.

---

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+O** | Open File | Open a .fxd file |
| **Ctrl+S** | Save | Save current file |
| **Ctrl+Shift+S** | Save As | Save with new name |
| **Alt+F4** | Exit | Close the application |

---

## Edit Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+Z** | Undo | Revert last change |
| **Ctrl+Shift+Z** | Redo | Reapply undone change |
| **Ctrl+C** | Copy | Copy selected nodes |
| **Ctrl+V** | Paste | Paste copied nodes |
| **Ctrl+X** | Cut | Cut selected nodes |
| **Delete** | Delete | Delete selected nodes |
| **Ctrl+A** | Select All | Select all nodes in scene |
| **Esc** | Clear Selection | Deselect all nodes |

---

## View Controls

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl++** | Zoom In | Zoom camera closer |
| **Ctrl+-** | Zoom Out | Zoom camera farther |
| **Ctrl+0** | Reset Zoom | Return to default zoom |
| **Ctrl+M** | Toggle Minimap | Show/hide minimap |
| **Ctrl+Shift+P** | Toggle Performance | Show/hide performance overlay |
| **F11** | Full Screen | Toggle fullscreen mode |
| **Ctrl+R** | Reload | Reload the visualizer |
| **F12** | Dev Tools | Open developer tools |

---

## Search & Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+F** | Find | Open search panel |
| **F** | Focus Node | Center camera on selected node |
| **Esc** | Close Search | Close search panel |

## Mouse Controls

While not keyboard shortcuts, these mouse controls are essential:

| Control | Action |
|---------|--------|
| **Left Click** | Select node |
| **Ctrl+Left Click** | Add to selection |
| **Left Drag** | Rotate camera |
| **Right Click** | Context menu |
| **Middle Drag** | Pan camera |
| **Mouse Wheel** | Zoom in/out |
| **Double Click** | Edit node (coming soon) |

---

## Context Menu Actions

Access these via **Right Click** on a node:

| Action | Shortcut | Description |
|--------|----------|-------------|
| Copy Node | Ctrl+C | Copy the node |
| Paste Node | Ctrl+V | Paste copied node |
| Delete Node | Delete | Remove the node |
| Focus on Node | F | Center camera on node |
| Expand All | - | Expand all children |

## Pro Tips

### Speed Navigation
- **Ctrl+F** → type node name → **Enter** → **Esc**
  - Fastest way to find and focus on a node

### Bulk Operations
- **Ctrl+Click** multiple nodes → **Delete**
  - Delete many nodes at once

### Screenshot Workflow
- **Ctrl+M** (hide minimap)
- **Ctrl+Shift+P** (hide performance)
- Position camera
- **Ctrl+Shift+P** (export PNG)

### Precision Work
- **F** to focus on node
- **Ctrl++** for closer view
- **Right Click** for options
- **Ctrl+Z** if you make a mistake
